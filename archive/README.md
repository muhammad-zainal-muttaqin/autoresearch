# Archive

This folder holds non-default material that is kept for reference, audit, and historical context.

Contents include:
- consolidated research notes
- legacy helper scripts
- archived model variants
- the analysis notebook

Default rule:
- do not start here for normal experimentation
- use `train.py`, `prepare.py`, `program.md`, `results.tsv`, and `logs/` first

Use this folder only when:
- you need historical context
- you need to inspect an old branch or experiment
- a human explicitly asks for a non-default formulation
